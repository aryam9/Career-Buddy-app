CareerBuddy is a comprehensive job application management tool designed to help users track their career journey efficiently. Whether you are actively job hunting or keeping a record of past applications, CareerBuddy provides an organized and intuitive platform to manage your job applications, interviews, offers, and rejections in one place.

CareerBuddy leverages modern and scalable technologies to provide a smooth and reliable user experience. The backend and database functionalities are powered by Supabase, which offers secure data storage, real-time updates, and authentication services. The frontend is built using HTML, CSS, and JavaScript to deliver an interactive and user-friendly interface. This combination allows CareerBuddy to be responsive, efficient, and secure, ensuring users can manage their job applications with ease and confidence.
